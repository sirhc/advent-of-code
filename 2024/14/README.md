# Day 14: Restroom Redoubt

https://adventofcode.com/2024/day/14

## Part One

This problem is suspiciously straight forward. I didn't even need to do any simulation. The position of each robot at
time `t` is a simple calculation. What's the catch?

```
❯ perl robots.pl 101 103 100 input | tail -4
{
    quadrants     => { 1 => 118, 2 => 128, 3 => 110, 4 => 133 },
    safety_factor => 220971520,
}
```

## Part Two

Wait, what? Okay, that's funny. Fortunately, I wrote my code in part one to print out the map, because I like to
visualize these things. And, since I wrote it to only calculate the final position, I could comment out the quadrant
code and just use it as-is.

I also have Vim configured to move to the next buffer when the down arrow is pressed. So it was pretty easy to skim
through the output.

```
❯ for i in {001..100}; do perl robots.pl 101 103 $i input > $i.out; done

❯ vim *.out
```
